# アイスが売れると事故が増える？

勉強をがんばったら成績が上がったとしよう。普通「成績が上がったのは勉強をしたおかげ」と考えるだろう。さらに、親や先生に「成績を上げたかったら勉強しろ」と口うるさく言われていることだろう。「成績が良い人はみんな勉強している。だから成績を上げたかったら勉強しろ」という言葉を聞いた時、「うるさいなぁ」とは思っても「本当かな？」と疑ったりはしないであろう。

では、「成功した人はみんな高い時計をしている。だから成功したければ高い時計を買え」という言葉はどうだろうか。一瞬「そうかもなぁ」と思うが、「あれ？」ともやもやする気持ちになるだろう。「朝ごはんをしっかり食べている学生と、食べていない学生では、食べている学生の方が成績が良かった」という「事実」があったとしよう。ここから「朝ごはんをしっかり食べると頭が良くなる」と結論付けて良いのだろうか？
