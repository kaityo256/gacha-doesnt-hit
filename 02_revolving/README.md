## 数値シミュレーションとは

本書のテーマは数値シミュレーションである。数値シミュレーションはシミュレーションの一種だ。シミュレーションとは「予め定めたルールに従う世界の振る舞いを再現、予想すること」である。例えば都市や国を運営するゲームはシミュレーションゲームと呼ばれるジャンルである。飛行機のパイロットは、フライトシミュレーターで訓練を行う。フライトシミュレーターは実機と同じ計器や操縦桿が用意され、同じ感覚で操縦できる。例えば悪天候での飛行や、エンジントラブルなど危険な状況を再現し、操縦やトラブル対応の訓練ができる。避難訓練などもシミュレーションの一種だ。

数値シミュレーションとは、シミュレーションを数値的に行うことだ。
