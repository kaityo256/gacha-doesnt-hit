# 風邪はなぜ流行る？

## ゾンビパニック

あなたの町にゾンビが現れた。ゾンビは一週間で活動を停止するが、その間に噛まれた人はゾンビになってしまう。人がゾンビになってから活動停止するまでに、平均で$r$人に噛みついてゾンビにしてしまう。もともと1万人いた町のうち、100人がゾンビになったとする。ゾンビが全員活動停止するまでにどれくらいの被害が出るだろうか？小数点以下は切り捨てとしてシミュレーションしてみよう。

最初に$r=0.1$としてみよう。つまり、10匹いるゾンビが活動を停止するまでに1人だけが被害にあう。最初に100匹のゾンビがいるのだから、最初の1週間で10人がゾンビとなり、最初にいた100匹が活動を停止する。次に10匹のゾンビが1人をゾンビにして活動を停止する。最後の1匹は他の誰もゾンビにすることなく活動停止する。ゾンビになったのは最初のゾンビも含めてトータルで111匹、生存者は9789人だ。

$r=0.5$の場合、つまり2匹のゾンビに対して被害者が1人でるような状態を考える。最初に100匹のゾンビがいて、50人がゾンビになる。次の週は25人が、次は12人・・・と被害者は半分になっていく。全てのゾンビが活動を停止するのは8周目で、それまでに197人が被害にあうことになる。$r=0.9$の場合は、事態が収束するのに29週、被害者は863人、生存者は9037人となる。

しかし、$r=1$になった瞬間、状況が大きく変わる。100匹のゾンビは、100人をゾンビにして活動を停止する。次の週も100匹のゾンビがいて、100人をゾンビにする。活動中のゾンビの数が減らないため、そのまま全員がゾンビになるまでゾンビパニックは終わらない。最終的に99週経過後に、生存者ゼロで事態は収束(？)する。
