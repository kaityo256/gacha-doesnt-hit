# 風邪はなぜ流行る？

## ゾンビパニック

あなたの町にゾンビが現れた。ゾンビは一週間で活動を停止するが、その間に噛まれた人はゾンビになってしまう。人がゾンビになってから活動停止するまでに、平均で$R$人に噛みついてゾンビにしてしまう。もともと1万人いた町のうち、100人がゾンビになったとする。ゾンビが全員活動停止するまでにどれくらいの被害が出るだろうか？この時の被害はパラメータ$R$にどのように影響するだろうか？小数点以下は切り捨てとしてシミュレーションしてみよう。

最初に$R=0.1$としてみよう。つまり、10匹いるゾンビが活動を停止するまでに1人だけが被害にあう。最初に100匹のゾンビがいるのだから、最初の1週間で10人がゾンビとなり、最初にいた100匹が活動を停止する。次の週に10匹のゾンビが1人をゾンビにして活動を停止する。最後の1匹は他の誰もゾンビにすることなく活動停止する。ゾンビパニックは3週目には収束し、ゾンビになったのは最初のゾンビも含めてトータルで111匹、生存者は9789人だ。

$R=0.5$の場合、つまり2匹のゾンビに対して被害者が1人でるような状態を考える。最初に100匹のゾンビがいて、50人がゾンビになる。次の週は25人が、次は12人・・・と被害者は半分になっていく。全てのゾンビが活動を停止するのは7周目で、それまでに197人が被害にあうことになる。$R=0.9$の場合は、事態が収束するのに28週、被害者は863人、生存者は9037人となる。

ここまではゾンビが全て活動停止した後に生存者がいる、いわゆる「生存エンド」であった。しかし、$R=1$になった瞬間、状況が大きく変わる。100匹のゾンビは、100人をゾンビにして活動を停止する。次の週も100匹のゾンビがいて、100人をゾンビにする。活動中のゾンビの数が減らないため、そのまま全員がゾンビになるまでゾンビパニックは終わらない。最終的に99週経過後に生存者はゼロ、すなわち全滅エンドを迎える。

$R$が1より大きい場合、活動するゾンビの数がどんどん増えていくため、必ず全滅エンドとなる。例えば$R=1.1$の場合、最初は100匹だったゾンビは、次週は110匹、3周目は121匹、133匹、146匹と急速に増えていき、最終的に25週目に生存者ゼロ、すなわち全滅エンドを迎える。$R=2.0$の場合は活動するゾンビが200匹、400匹と倍々ゲームで増えていき、6週目に全滅する。

以上の状況をまとめると、パラメータが$R<1$の時には生存エンド、パラメータが$R\geq 1$の時には全滅エンドとなる。また、$R=1$の場合と$R>1$の場合は両方とも全滅エンドだが、その性質が異なる。$R<1$の時にはゾンビの数が指数関数的に減衰していく。一方で$R>1$の時には逆に指数関数的に増加していく。$R$がちょうど$1$の時には、ゾンビは増えもしないし減りもしない。生存者が時間に比例して減っていく。このように、あるパラメータに「しきい値」があり、その「しきい値」をまたぐと大きく状況が変わるような現象を「相転移」と呼ぶ。相転移は、ミクロには同じルールに従って振る舞うが、パラメータが少し変化すると大きく性質が変わるような現象を言う。身近な例では一気圧下では水が摂氏0度で凍り、100度で沸騰するのも相転移の一種である。また、このような相転移の「しきい値」を「相転移点」と呼ぶ。ゾンビパニックの場合は$R=1$が相転移点であり、水の場合は0度や100度が相転移点である。一般に相転移点では不思議なことがいろいろ起きる。

先ほどのゾンビの例は、ちょっと考えれば「$R=1$を境に性質が大きく変わるだろうな」と予想できるであろう。では、もう少し非自明な例を考えよう。

道路が碁盤の目状になっているような都市、たとえば札幌を考えよう。その都市を車で横断したいが、大雪が降ったために道が所々通行止めになってしまった。無事な道だけを通ってこの都市を横断できる確率、横断可能確率を考えてみよう。今、$L \times L$の碁盤の目状の道を考える。このうち、通行可能な道の割合、つまり通行可能確率が$p$%であるとしよう。この時、横断可能確率$P(p)$を通行可能確率$p$の関数として求めよう。例えば通行可能確率が50%の時、それぞれの道に対してコインを投げ、表なら通過可能、裏なら通過不能としていく。全ての道に対して通過可能・不能を決めたあと、通過可能な道だけを通って左から右にいけるかどうかを調べる。これを同じ$p$の値に対してなんども繰り返し、横断に成功した回数を数える。例えば100回繰り返して25回通過できたなら、横断可能確率$P$は25%である。こうして求めた横断可能確率$P(p)$はどんな形をしているだろうか？

すぐにわかることとして、全ての道が通過不可能、つまり通行可能確率が0%の場合には絶対に横断できないため、横断可能確率は0%である。一方、全ての道が通過可能、つまり100%の場合には、絶対に横断可能であるから横断可能確率は100%である。さらに、$p$が大きくなればなるほど横断可能確率も増えるであろうことも予想できる。では、$0 < p < 1$の間に何が起きるか予想できるだろうか？

道の通行確率が50%、つまり$p=0.5$に達するまでは、横断可能確率$P$はほとんどゼロ、つまり横断は不可能である。一方、$p=0.5$を超えた瞬間に横断可能確率は、ほぼ100%になる。つまりこの都市は、道の通行可能確率が、ある「しきい値」$p=0.5$に達する前と後で大きく性質が変わる。通行可能確率が50%よりもわずかでも小さい(例えば49.99%)と、この都市は横断不可能だ。一方、50%を少しでも超えると(例えば50.01%)、この都市は横断可能となる。実は、これも水の沸騰と同じ、相転移現象の一種であり、特にこの現象をパーコレーション転移と呼ぶ。パーコレーションとは「浸透」という意味だ。コーヒーが好きな人なら、パーコレーターという器具を知っているかもしれない。パーコレーターは英語のパーコレート(percolate)という言葉に由来する。これは「液体が何かにゆっくりと染みていく、濾過される」という意味がある。パーコレーション転移とは、左から水を流した時、水が途中で止まる相(非パーコレーション相)から、水が通過可能な道を通って右へと染み出していく相(パーコレート相)への相転移である。では、どうしてほんの少しのパラメータの変化で、そんなに大きな変化が起きるのかを調べて見よう。

まず、ある交差点から、通行可能な道を通って別の交差点に行ける時、これらの交差点は「同じグループ」に属すと定義しよう。交差点Aから交差点Bに到達可能で、かつ交差点Bから交差点Cに到達可能なら、「友達の友達は友達」理論により、交差点A、B、Cはすべて同じグループだ。道の通行確率が非常に小さい時、それぞれの交差点からせいぜい隣にいけるか行けないか、すなわち交差点は「ぼっち」になる。しかし、道の通行確率が増えるにつれて「友達」が増えていき、グループが大きくなっていく。通行可能確率が50%に近づいてくると、大きなグループがうまれ、それぞれのグループがぶつかりそうになる。そして、通行可能確率が50%を超えた瞬間、街の中にあった友達グループが一気につながり、都市全体を覆う大きなグループができる。都市が横断可能かどうかは、都市の端から端に達するような大きなグループができているかどうかと等価である。
