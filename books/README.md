# ブックガイド

## 「数学ガール シリーズ (結城浩 (著)、SBクリエイティブ)」

数学に魅せられた高校生達のドラマを瑞々しい筆致で描いている本です。魅力的な登場人物の会話を追いかけているうちに数学の面白さにハマっていることでしょう。

## 「闘うプログラマー (パスカル ザカリー (著), 山岡 洋一 (翻訳)、日経BP)」

マイクロソフトでWindows NTを開発した伝説のプログラマー「デイヴィッド・カトラー」の伝記のような本です。巨大企業の命運をかけたプロジェクトは、超人的な努力により完遂されました。ビッグプロジェクトの面白さと怖さを教えてくれます。

## 「ザ・ゴール　シリーズ　(エリヤフ・ゴールドラット (著), 三本木 亮 (訳), ダイヤモンド社)」

良かれと思ってやることが、なぜか全体の足を引っ張ることがあります。「いま何が目的で、そのためには何をすべきか、何をすべきでないか」を考えさせてくれる本です。ストーリーもとても面白いです。

## 「ゼロから学ぶPythonプログラミング (渡辺宙志 (著) 講談社サイエンティフィク)」

大学のプログラミングの講義ノートが元になった教科書です。本書で出てきたシミュレーションを多数扱っています。興味があれば、この本を参考に実際にプログラムを書いてみると良いでしょう。

## 「はやぶさ、そうまでして君は〜生みの親がはじめて明かすプロジェクト秘話 (川口 淳一郎 (著) 宝島社)」

世界で初めて地球重力圏外にある天体の固体表面に着陸し、地球にサンプルを持ち帰った宇宙探査機「はやぶさ」の本です。このプロジェクトの実行にはたくさんの困難がありました。「はやぶさ」の本は多数出版されていますが、プロジェクトの責任者である川口淳一郎氏の視点から書かれた本書を読むと、当時の興奮が伝わってきます。

## 「ニュートリノの夢 (小柴 昌俊 (著) 岩波ジュニア新書)」

2002年にノーベル物理学賞を受賞した小柴昌俊氏による回顧録のような本です。どのようにしてノーベル賞を受賞するような研究に成功したのかを平易に説明しています。物理学者が書いた本ですが、読みやすい内容になっています。

## 「論文捏造 (村松 秀 (著) 中公新書ラクレ)」

世界で最も有名な研究所の一つであるベル研究所、その若きホープにより次々と発表された素晴らしい成果に世界は驚きます。「ノーベル賞確実」とまで言われたそれらの実験結果は、実は捏造されたものでした。科学には光もあれば闇もあります。その闇について書かれた本です。

## 「ライト、ついてますか: 問題発見の人間学　 (ドナルド・C・ゴース (著), G.M.ワインバーグ (著), 木村 泉 (翻訳) 　共立出版)」

何か問題がある時に、いくらしつこく注意書きを貼り出してもまったく効果がない時があります。その時、ちょっと発想を変えることで問題が解決するかもしれません。少し古い本ですが、現在でも十分に読む価値のある本です。
