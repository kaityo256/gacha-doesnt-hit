# ガチャの沼へようこそ

「ガチャ」という言葉を聞いたことがあるでしょう。もともとは、「カプセルトイ」と呼ばれる小型の自動販売機から来た言葉です。カプセルトイは、硬貨を入れてレバーを回すと、カプセルに入ったおもちゃなどが出てくる仕組みになっています。レバーを回すときに「ガチャガチャ」と音が鳴ることから、「ガチャ」や「ガチャガチャ」と呼ばれるようになりました。なお、バンダイは「ガシャポン」や「ガチャガチャ」という名前を商標登録しています。

カプセルトイの面白さは、お金を入れてカプセルが出てくるまで、中に何が入っているかわからないところにあります。さらに、たまに「シークレット」と呼ばれる、一覧には載っていない特別な種類が入っていることもあります。そのため、「次こそ欲しいものが出るかもしれない」と思って、つい何度も挑戦したくなります。

やがて、このような仕組みはソーシャルゲーム、いわゆるソシャゲにも広がりました。ソシャゲでは、お金を払うと、キャラクターやアイテムがランダムに手に入ることがあります。この仕組みも「ガチャ」と呼ばれています。

ガチャで手に入るキャラクターやアイテムには、「ノーマル」「レア」「スーパーレア」「ウルトラレア」などのように、手に入りやすさを表す「レア度」が設定されています。レア度が高いものほど手に入りにくく、多くの人がそれを目当てにガチャを引きます。その結果、欲しいキャラクターやアイテムが出るまで何度も課金してしまう、いわゆる「廃課金」が問題になることもあります。

多くの場合、レアアイテムやレアキャラクターが出る確率は公開されています。しかし、実際にガチャを引いてみると、「確率は◯％のはずなのに、なかなか当たらない」と感じることがあります。これは、確率が公開されていても、短い回数では結果にばらつきが出るからです。

以下では、このようなガチャの仕組みについて、シミュレーションを使って調べてみましょう。

## ガチャでレアが当たる確率

ガチャを一度回すと、あるアイテムが出る確率が1%だったとしましょう。運が良い人はいきなり最初に当ててしまう人もいるでしょう。運が悪い人は1000回回しても当たらないかもしれません。このアイテムを当てるには平均何回ガチャを回せばよいでしょうか？また、どのくらい運が悪いと1000回回しても当たらないのでしょうか？ちょっと計算してみましょう。




## アイドルのイベント

あるアイドルが新曲のリリースを記念してイベントを行った。そのシングルCDの購入者に対してポスターをプレゼントする。ポスターは全部で44種類あり、もらえるポスターはランダムである。このポスターの44種類すべてコンプリートしたら、特別なイベントに招待します、というものだ。最初にCDを購入した時、ポスターが1枚もらえる。2枚目を購入した時に、同じポスターを当ててしまう確率は44分の1だ。手元のポスターの書類が増えるほど、次にもらえるポスターが手持ちと「かぶる」可能性が高くなっていく。例えば、いま手元に22種類のポスターを持っているなら、次のCDがかぶる確率は50%だ。さて、ポスターをすべてコンプリートするには、平均で何枚のCDを購入すればよいだろうか？答えを読む前に、まず予想してみてほしい。予想できただろうか？では次に進もう。

この問題は厳密解が存在するが、まずはシミュレーションしてみよう。まずは1から44の数字を紙に書いておく。44つの目があるサイコロを用意して、サイコロを振っては出た目の数字を消す。サイコロを振った数と、消した数字の数の関係を調べ、最後の数字を消した時の「サイコロを投げた回数」を記録すればよい。コードも非常に短いので図Xに掲載してみよう。ひたすらCDを購入し、44種類のポスターをコンプリートするまでに何枚必要だったかを記録する試行を1万回繰り返し、その平均を表示するコードだ。

```py
import random

total = 0
for i in range(10000):
  cd = []
  n = 0
  while len(set(cd)) < 44:
    i = random.randint(1,44)
    cd.append(i)
    n += 1
  total += n

print(total/10000)
```

図：CDのコンプリートに必要な枚数を調べるモンテカルロシミュレーションコード。

実行してみると、毎回結果が変わるが、およそ190～193という値が出るであろう。このポスターコンプリート問題は厳密解も計算可能なので、その説明もしておこう。$n$種類のポスターを所持している時に、新しいポスターを得るのに必要なCDの数を$C(n)$としよう。例えば最初は一枚もポスターを持っていないので$n=0$である。したがって、CDを1枚購入すれば必ず新しいポスターが手に入るので、$C(0)=1$である。次に、1枚のポスターを持っている状態では、同じポスターがかぶる確率は44分の1だ。したがって、新しいポスターがもらえる確率は$43/44$となる。その逆数が「新しいポスターを手に入れるのに必要なCD購入数」となるので、$C(1) = 44/43$となる。こうして、一般項として$C(n) = 44/(44-n)$が得られる。これを$C(0)$から$C(43)$までの和を計算すれば必要な総CD枚数の期待値の厳密解が得られる。計算してみると約192.4枚となることがわかる。

さて、この192枚という数字は、先ほどの予想と比べてどうだっただろうか？「思ったより少ないな？」と感じた人も多かったと思う。もっと1000枚とか2000枚とか必要な、あくどい商売を想像したのではないだろうか。実は、ギャンブルは「プレイヤーがハマる」ように、「プレイヤーにほどよく勝たせる」必要がある。たとえばパチンコ屋に入った時に、誰一人として大当たりをひいていないと、お客さんはすぐに諦めてしまう。ぱっとホールを見渡した時、最低でも誰か一人は常に大当たりを引いていることが望ましい。そし台の前に座ってしばらくして隣の台の人が大当たりを引いたら、「自分もそろそろ当たるかも」と思って止め時を見失うであろう。

さらに、ギャンブルにはもう一つ重要な要素がある。それは、「ここで諦めたらこれまでの投資が無駄になってしまう」という気持ちだ。これを調べるために、「CDを$n$枚購入している状態で、もう一枚CDを購入した時、新しいポスターを手に入れる確率」を計算してみよう。ある特定のポスターに注目する。$n$枚CDを購入したとして、そのポスターをまだ引けていない確率$P_1$は$(43/44)^n$である。ポスターは44種類あるから、$n$枚のCDを購入した時点で、まだ手に入れていないポスターの種類は先程の値の44倍、つまり$44P_1$である。この状態で1枚CDを購入すると、44種類のポスターのどれかが手に入る。手に入れていないポスターを手に入れる確率は$44P_1/44$、つまり$P_1 = (43/44)^n$に等しい。要するにCDを購入した時に、新たなポスターを手に入れる確率は、CDの購入枚数に対して指数関数的に減少する。

ポイントは、「新しいポスターを手に入れる確率」は最初は高いことだ。手元にあるポスターの種類が少ない時には「かぶる」確率が低いので、どんどん新しいポスターが手に入ることになる。しかし、徐々に「かぶる」確率が増えていく。100枚くらいCDを購入すると、ポスターのコンプ率は90%程度になる(図)。つまり、後4枚か5枚だ。「あと少し」と思うことだろう。しかし、先ほど見たように、44枚のポスターを揃えるためには200枚近いCDを購入しなければならない。つまり、「コンプ率90%でちょうど道半ば」なのである。

さて、ここで100枚購入した人の気持ちを考えてみよう。手元には100枚、40種類のポスターがある。どんどん新しいポスターを獲得する確率が下がっており、嫌な予感がする。しかし44種類を集めないと特別なイベントには参加できない。ここで諦めてしまったら、100枚同じCDを購入した「投資」が無駄になってしまう。あと少し、あと少しだから・・・こうして新たなCD購入へと向かうことになる。

このように「この時点で撤退したら回収できない投資」のことを「サンクコスト(埋没費用)」と呼ぶ。プロジェクトが明らかに失敗しているのに、「いま諦めたらこれまでに投資した費用がすべて無駄になる」という気持ちからなかなか撤退できない気持ちを「サンクコスト効果」と呼ぶ。
