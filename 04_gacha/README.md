# ガチャの沼へようこそ

「ガチャ」という言葉を聞いたことがあるだろう。もともとは「カプセルトイ」と呼ばれる小型の自動販売機に由来する。カプセルトイは硬貨を入れてレバーを回すと、カプセルに入ったおもちゃなどが出てくるものだ。レバーを回す際にガチャガチャ音がなるので、ガチャガチャなどと呼ばれるようになった。バンダイは「ガシャポン」や「ガチャガチャ」という名前で商標登録をしている。お金を入れてカプセルが出てくるまで何が出てくるかわからないという楽しみがあり、たまに「シークレット」と呼ばれる、表に表示されていない種類が入っていることがあり、何度もお金を入れたくなる装置である。

そのうち、特にソーシャルゲーム(ソシャゲ)などで、購入した時にもらえるアイテムがランダムで決まるようなものも「ガチャ」と呼ばれるようになった。アイテムには「ノーマル」「レア」「スーパーレア」「ウルトラレア」などと「レア度」が決まっており、レアなアイテムを求めて何度も課金してしまう、いわゆる「廃課金」が問題となった。以下では、そのようなガチャについてのシミュレーションしてみよう。
