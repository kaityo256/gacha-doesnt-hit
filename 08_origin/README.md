# 学問との出会い

たまに、「どうして研究者になったのですか？」と聞かれます。何かドラマチックなきっかけがあると良いのですが、私の場合は単純でした。父が物理学者だったのです。父の専門は高エネルギー実験と呼ばれる分野で、この分野では日本は小柴昌俊氏、梶田隆章氏がノーベル物理学賞を受賞しています。父は小柴研出身であり、しかも父が助教授の時に梶田先生が学生で研究室にいたそうです。さらに父がアメリカにいた時に、後にノーベル物理学賞を受賞することになる南部陽一郎氏とも交流があり、晩ごはんの話題にノーベル物理学賞受賞者の名前が頻繁に出てくるような、ノーベル賞が身近といえば身近な家庭でした。私が小学生の頃、父は高エネルギー物理学研究所というところに勤めていましたが、いつもいそいそと研究所に向かう姿を見て、「研究所とは楽しいところに違いない」と思っていたものです。進路を父に相談した記憶はありませんが、小学生の頃には「将来の夢」は研究者でした。結局、小学生の頃から夢は変わらず、そのまま研究者となりました。後に私も物性研究所というところに勤めましたが、「研究所とは楽しいところ」というイメージそのままでした。研究は楽しいものです。本書を読んで、「研究ってなんだか楽しそうだな」と感じ、研究者を目指してみようと思ってくださる方が少しでも増えたなら、これほど嬉しいことはありません。
