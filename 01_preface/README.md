# はじめに

本書は、これから大学に進学しようとする高校生に向けて書いたものです。保護者や先生に「(良い)大学に行け」と言われ、なんとなく勉強してきたものの、大学とは何をするところなのか、大学で何を学ぶのか、イメージがぼんやりしている人も多いことでしょう。

例えば高校まではクラスに所属して毎朝ホームルームがありましたが、大学ではクラスの概念は薄く、ホームルームなどもありません。高校までは基本的に毎日授業がみっちりありましたが、大学では必修以外は自由に履修して良く、「空き時間」が出来たりします。さらに、大学では概ね学科を単位にカリキュラムが決まっており、カリキュラムによって学ぶことが違います。さらに言えば、同じ学科でも所属する研究室によってテーマが異なるので、卒業研究や修士論文研究の内容も全く違います。したがって、「大学で学ぶこと」を具体的に説明することはできません。

考えてみるとこれは不思議なことです。高校までは、学ぶことは概ね「学習指導要領」で決まっており、高校を卒業した人は「これとこれを学んだはず」という共通点があります。一方、大学では、卒業研究で所属する研究室が実験系か理論系かによってやることが大きく変わります。全員違うことを学んでいるのに、卒業時にはみんな同じ「学士号」「修士号」を受け取ることになります。

結局、大学では何を学ぶのでしょうか？なぜみんな違うことをしているのに、同じ資格を得るのでしょうか？大学を卒業した人に共通点はあるのでしょうか？そもそも大学は皆さんに何を求めているのでしょうか？
