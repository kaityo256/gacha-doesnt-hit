# はじめに

本書は、これから大学に進学しようとする高校生に向けて書いたものです。保護者や先生に「(良い)大学に行け」と言われ、なんとなく勉強してきたものの、大学とは何をするところなのか、大学で何を学ぶのか、イメージがぼんやりしている人も多いことでしょう。

中学校から高校に進学した時にはあまり生活は変わらなかったと思いますが、高校生活と大学生活はかなり違います。例えば高校まではクラスに所属して毎朝ホームルームがあったと思いますが、大学ではクラスの概念は薄く、ホームルームなどもありません。高校までは基本的に毎日授業がみっちりありましたが、大学では必修以外は自由に履修して良く、講義と講義の間に時間が空いたりします。さらに、大学では概ね学科を単位にカリキュラムが決まっており、カリキュラムによって学ぶことが違います。また、大学や学科にもよりますが、多くの理工学系の学部では、4年生になると研究室に配属され、何かのテーマについて研究を行い、その結果を卒業論文としてまとめることになります。研究室でどんな研究をするかは、配属された研究室の教員の専門によって異なります。例えば実験系の研究室なら実験を、理論系の研究室なら理論計算を、数値計算の研究室なら数値シミュレーションを行うことになるでしょう。

考えてみるとこれは不思議なことです。高校までは、学ぶことは概ね「学習指導要領」で決まっており、高校を卒業した人は「これとこれを学んだはず」という共通点があります。一方、大学では所属する研究室によってやることが大きく変わります。全員違うことを学んでいるのに、卒業時にはみんな同じ「学士号」という称号を受け取ることになります。結局、大学では何を学ぶのでしょうか？なぜみんな違うことをしているのに、同じ資格を得るのでしょうか？大学を卒業した人に共通点はあるのでしょうか？そもそも大学は皆さんに何を求めているのでしょうか？


実は、大学は「ただ高度な知識を身につけるところ」ではありません。研究活動を通じて「考える力」を身につけるところです。高校生までは基礎知識を身に着けることが勉強の主な目的でしたが、大学からはそれまでに身に着けた知識を総動員し、知らないことに立ち向かっていかなければなりません。

理論、実験、数値計算、研究テーマはそれぞれ違いますが、自らの頭で結果を解釈し、そして形にまとめてアウトプットする、そんな一連の流れをマスターし、「この人は自ら考えることができる人材ですよ」と、大学が認める、それが学士号であり、修士号です。

では、これまでに得た知識を総動員して結果を解釈するとは、具体的にどんなことをするのでしょうか？実験、理論、フィールドワークなど学問には様々な取り組み方がありますが、本書では数値シミュレーションを題材に、「考える」ことについて考えてみましょう。
