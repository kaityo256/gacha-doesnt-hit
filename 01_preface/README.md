# はじめに

本書は、これから大学に進学しようとする高校生に向けて書いたものである。親や先生に「(良い)大学に行け」と言われ、なんとなく勉強してきたものの、大学とは何をするところなのか、大学で何を学ぶのか、イメージがぼんやりしている人も多いだろう。特に、大学は高校までとはかなり仕組みが異なるので、さらにイメージしづらくなる。例えば高校まではクラスに所属して毎朝ホームルームがあったが、大学ではクラスの概念は薄く、ホームルームなどもない。高校までは基本的に毎日授業がみっちりあるが、大学では必修以外は自由に履修して良く、「空き時間」が出来たりする。さらに、大学では概ね学科を単位にカリキュラムがが決まっており、カリキュラムによって学ぶことが違う。もっと言えば、同じ学科でも所属する研究室によってテーマが異なるので、卒業研究や修士論文研究の内容も全く異なる。したがって、「大学で学ぶこと」を具体的に説明することはできない。

考えてみるとこれは不思議なことである。
