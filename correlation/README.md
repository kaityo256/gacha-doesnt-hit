# アイスが売れると事故が増える？

勉強をがんばったら成績が上がったとしましょう。この時、「成績が上がったのは勉強をしたおかげ」と考えるでしょう。さらに、保護者の方や先生方に「成績を上げたかったら勉強しろ」と口うるさく言われていることでしょう。「成績が良い人はみんな勉強している。だから成績を上げたければ勉強しろ」という言葉を聞いた時、「うるさいなぁ」とは思っても「本当かな？」と疑ったりはしないですよね？「成績が良い人はみんな勉強している。だから成績を上げたければ勉強しろ」という文章をもう少し見てみましょう。この文章は「Ｘしている人はＹしている。だからＹすればＸできる」という形をしています。

では、「成功した人はみんな高い時計をしている。だから成功したければ高い時計を買え」という言葉を聞いてどう思うでしょうか？この文章は、先程の「Ｘしている人はＹしている。だからＹすればＸできる」という文章と同じ形をしています。これを聞いて、一瞬「そうかもなぁ」と思うかもしれませんが、よく考えると「あれ？」ともやもやしませんか？

同じように、「朝ごはんをしっかり食べている学生と、食べていない学生では、食べている学生の方が成績が良かった」という「事実」があったとしましょう。ここから「朝ごはんをしっかり食べると頭が良くなる」と結論付けて良いのでしょうか？
